===========
awibmanager
===========

This package bundles dependencies, code, and scripts for managing the Ancient World Image Bank (AWIB), a project of the Institute for the Study of the Ancient World (ISAW), New York University.

awibmanager is written by Tom Elliott, for ISAW. See LICENSE.txt for copyright and licensing information.

Dependencies
---------------

	* Packages installed via homebrew (order significant)

		* exiftool (9.35)
		* python 2.7.5
		* Before Pillow:
			* libpng (1.5.14)
			* libtiff (4.0.3)
			* libjpeg (jpeg 8d)
			* webp (0.3.1)
			* littlecms (little-cms 1.19)

	* Python modules:

		* elementtree (1.2.6-20050316)
		* flickrapi (1.4.2)
		* lxml (3.2.3)
		* nose (1.3.0)
		* Pillow (2.2.1)
		* pip (1.4.1)
		* setuptools (0.9.8)
		* simplejson (3.3.1)
		* wsgiref (0.1.2)
